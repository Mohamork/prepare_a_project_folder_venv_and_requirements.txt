# prepare_a_project_folder_venv_and_requirements.txt
A script to prepare a Python project by creating a project folder, venv, and installing packages.

##### I have no requirements.txt
No problem. The script will create a project folder and venv. No packages get installed.

##### I have a requirements.txt but it's empty
No problem. The script will create a project folder and venv. No packages get installed.

##### I have a requirements.txt with content
The script will create a project folder, venv and install packages from the requirements.txt file.
